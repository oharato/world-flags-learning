# 画面設計書（ワイヤーフレーム）

本ドキュメントは、国旗学習アプリの各画面のレイアウトと要素を定義します。

---

## 1. トップページ

シンプルなエントリーポイント。ユーザーをクイズモードか学習モードに誘導します。

```
+------------------------------------------------------+
|                                                      |
|                国旗学習アプリ (仮)                   |
|                                                      |
+------------------------------------------------------+
|                                                      |
|                                                      |
|           +--------------------------+               |
|           |      クイズに挑戦する    |               |
|           +--------------------------+               |
|                                                      |
|           +--------------------------+               |
|           |      国旗を学習する      |               |
|           +--------------------------+               |
|                                                      |
|                                                      |
+------------------------------------------------------+
```

---

## 2. クイズ設定画面

クイズを開始する前の設定を行います。

```
+------------------------------------------------------+
| < トップページに戻る                                 |
+------------------------------------------------------+
|                                                      |
| ニックネームを入力:                                  |
| +--------------------------------------------------+ |
| | Guest123                                         | |
| +--------------------------------------------------+ |
|                                                      |
| クイズ形式:                                          |
| (o) 国旗を見て国名を選ぶ                             |
| ( ) 国名を見て国旗を選ぶ                             |
|                                                      |
| 出題範囲:                                            |
| [ 全世界 v ]  (ドロップダウン)                       |
|                                                      |
|           +--------------------------+               |
|           |          スタート        |               |
|           +--------------------------+               |
|                                                      |
+------------------------------------------------------+
```

---

```
|           +--------------------------+               |
|                                                      |

|           +--------------------------+               |
|           |         選択肢 C         |               |
|           +--------------------------+               |
|                                                      |
|           +--------------------------+               |
|           |         選択肢 D         |               |

+------------------------------------------------------+
```

```
|                                                      |

|           +--------------------------+               |
|           |      ランキングを見る    |               |
|           +--------------------------+               |
|                                                      |
|           +--------------------------+               |
|           |      トップに戻る        |               |
|           +--------------------------+               |
|                                                      |
+------------------------------------------------------+
```

```
+------------------------------------------------------+
```

---

## 6. 学習モード画面

フラッシュカード形式で国旗を学習します。

```
+------------------------------------------------------+
| < トップページに戻る                                 |
+------------------------------------------------------+
|                                                      |
| +--------------------------------------------------+ |
| |                                                  | |
| |                [国旗の画像]                      | |
| |                                                  | |
| |           (クリックで詳細表示)                   | |
| |                                                  | |
| +--------------------------------------------------+ |
|                                                      |
|      [< 前へ]         [次へ >]                      |
|                                                      |
+------------------------------------------------------+
```
*カードクリック後*
```
+------------------------------------------------------+
| < トップページに戻る                                 |
+------------------------------------------------------+
|                                                      |
| +--------------------------------------------------+ |
| | 国名: 日本                                       | |
| | 首都: 東京                                       | |
| |                                                  | |
| | [簡単な地図]                                     | |
| |                                                  | |
| | 国旗の成り立ち: ...                              | |
| | 国の概要: ...                                    | |
| +--------------------------------------------------+ |
|                                                      |
|      [< 前へ]         [次へ >]                      |
|                                                      |
+------------------------------------------------------+
```
